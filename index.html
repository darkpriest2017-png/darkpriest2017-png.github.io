🧬 О проекте:

Юлия — это локальный виртуальный ИИ-компаньон с характером и навыками UI/UX-дизайнера. Проект создан для совместной разработки интерфейсов, генерации творческих идей и лампового общения. Юля обладает долговременной памятью, анализирует контекст отношений и полностью управляется голосом, используя современные ИИ-технологии на базе локального графического процессора.

  • Мозг проекта: GigaChat API (в процессе подготовки к переходу на полностью локальные LLM через Ollama).
  • Долговременная память: Векторная база данных ChromaDB для хранения контекста бесед и общих воспоминаний.
  • Аппаратное ускорение: Вычисления перенесены на ядра Asus ROG Strix RTX 3060 12GB с помощью технологии DirectML.
  • Слух и Речь: Голосовое управление реализовано на базе встроенных модулей распознавания и генерации речи (STT/TTS).

Мурчик — Кибер-взломщик аналоговых систем
